About CollabOS

The basics

CollabOS is an "Operating System" made on Scratch. Scratch is a website made primarily for kids where people can drag around "blocks" on the screen to create code. The CollabOS project is a collaboration between many scratch users to create and add features to an OS simulator. Many people have contributed to CollabOS, but some have since left. CollabOS is inspired by OS wars on scratch, but everyone works together. Some members of the CollabOS project are pushing the limits of scratch, doing things such as creating encrypted chat rooms and cloud systems.

How CollabOS was started

Back in mid- to late-2019, Scratcher `elinero` created a forum thread on the Scratch Collaboration Forums, looking to recruit members for a project inspired by OSWars called CollabOS, where people would sign up to add new features to an OS simulator on Scratch. Membership was relatively low, but soon rose very quickly, with 14 collaborators within the project's official development studio, ranging from `mlcreater` to `Prokube` and `wamu_m`. Many more had contributed to the project, but have since left.

Notable achievement's

Many things have been accomplished during the CollabOS project, some big and some small. Some of the bigger things that have been made include:

  • A full cloud login system
  • An encrypted chat room
  • A programming language
  • A small video sharing platform
  • Multiple good logo's
  • And more!
