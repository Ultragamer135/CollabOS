About CollabOS

CollabOS is an "Operating System" made on Scratch. Scratch is a website made primarily for kids where people can drag around "blocks" on the screen to create code. The CollabOS project is a collaboration between many users to create and add features to an OS simulator. Many people have contributed to CollabOS, but some have since left. The CollabOS project was started by scratch user "elinero" in mid- to late-2019.

Notable achievements

Many things have been accomplished during the CollabOS project, some big and some small. Some of the bigger things that have been made include:

  • A full cloud login system
  • An encrypted chat room
  • A programming language
  • A small video sharing platform
  • Multiple good logo's
  • And more!
