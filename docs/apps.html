Apps

  • EvolutionStore

    • EvolutionStore is an "app store" that currently has no use, but is planned to allow adding apps to the homescreen.

  • CMD

    • CMD is a command line interface for CollabOS.

  • Notepad

    • Notepad is a text editor that also allows for formatting text.

  • Bratch editor

    • Bratch editor is an app that allows for editing Bratch code.

  • Citrus

    • Citrus is a work-in-progress chat app.

  • Glint

    • Glint is an animation sharing platform for small and basic animations.
